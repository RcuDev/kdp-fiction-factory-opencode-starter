# Contributing

This repository is a public starter version of a commercial fiction-production system.

Issues, typo fixes and small documentation improvements are welcome. Please do not submit large rewrites that attempt to recreate the paid product inside the starter repository.

Before opening a pull request:

- keep the starter simple;
- do not add premium workflows such as SCEN, Word Budget, KDP export or specialist agents;
- preserve the personal-use license;
- keep examples realistic and beginner-friendly.
